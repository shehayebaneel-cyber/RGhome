# RG Home Aley — CLAUDE.md

## What this is
An online store + admin for **RG Home**, a home-appliances & consumer-electronics
retailer in the Old Souk, Aley, Lebanon (owner: Ghassan Rachid, on Instagram
[@rghome_aley](https://instagram.com/rghome_aley) since 2020). Customers browse a
bilingual (Arabic/English) catalogue and place orders; the owner manages products
and orders from an admin dashboard. Payment is **Cash on Delivery** / reserve-in-store
for now — online card payment is a later phase.

## Stack & layout
- `web/` (Vite + React 18 + TypeScript + Tailwind v4 + React Router) — dev port 5173
- `server/` (Express + Prisma + Neon Postgres, `tsx` for dev) — port 4010
- Deploy: **Render** blueprint (`render.yaml`) — `rghome-api` (Node) + `rghome-web` (static)
- DB: Neon Postgres (`us-east-1`)

## Commands
- Dev: `cd server && npm run dev`  +  `cd web && npm run dev`
- Typecheck: `npx tsc --noEmit` in both `web/` and `server/`
- DB push: `cd server && npx prisma db push` (Neon may need 2–3 retries to wake)
- Seed: `cd server && npm run db:seed`
- Web build: `cd web && npm run build`
- NOTE: this machine blocks npm postinstall scripts. After `npm install` you must run
  `npx prisma generate` (server) and approve/run esbuild's binary (web) manually.

## Brand
- Voice: trusted local showroom — honest, helpful, neighbourly (not big-box).
- Tokens: see `web/src/index.css` `:root` — **brand blue accent** (#1E6BE0) on neutral
  white/slate surfaces; navy (#123A6B) for logo/headings; blue is an ACCENT, not the field.
  Light + dark themes both defined. `--brand-deep` = the always-dark navy surface
  (offer band / footer / admin sidebar) — never use `--navy` as a background (it flips
  to near-white in dark mode).
- Bilingual: `web/src/lib/i18n.tsx` — `t(en, ar)`; toggling to Arabic sets `dir="rtl"`.
- Images are currently **SVG vector art** (`web/src/components/Art.tsx`) — swap for real
  product photos when supplied. Logo is recreated as SVG pending the real logo file.

## Domain rules
- Only the **Samsung 12,000 BTU Inverter AC = $468** price is confirmed real; all other
  products/prices are realistic placeholders seeded for demo (`server/prisma/seed.ts`).
- Prices are whole USD integers.
- Delivery is **free within Aley** (`deliveryFee = 0`); orders never take online payment yet.
- Admin auth: JWT in an httpOnly cookie (`rg_admin`). In production the cookie is
  `SameSite=None; Secure` because web and API are separate Render origins.
- Categories are fixed slugs: ac, tv, oven, fridge, wash, coffee, small.
- Contact: phone 05 556 970 / 70 556 970 · rghome.aley@hotmail.com · Debbas St, Old Souk, Aley.
